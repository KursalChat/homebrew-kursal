## What is Kursal?

Kursal is a **peer-to-peer**, **end-to-end encrypted** messaging platform. No servers, no tracking. Just your private communications.

## Installation:

Check the [prototype repository](https://kursal.chat/repository) for all the informations and the installation steps.

To install with brew (what this repository hosts), run this:

```bash
brew install KursalChat/Kursal/kursal
```
